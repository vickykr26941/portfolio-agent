SETUP INSTRUCTIONS FOR NEW ENVIRONMENT
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. GET CLOUDFLARE API TOKEN
   → https://dash.cloudflare.com
   → Account → API Tokens
   → Create Custom Token
   → Set: Cloudflare Workers AI (Edit)
   → Copy token

2. GET CLOUDFLARE ACCOUNT ID
   → Same dashboard page
   → Right sidebar shows Account ID
   → Copy account ID

3. CONFIGURE ENVIRONMENT
   $ cp .env.example .env
   $ nano .env  # Edit with your values
   
   Add:
   CLOUDFLARE_API_TOKEN=your_token_here
   CLOUDFLARE_ACCOUNT_ID=your_account_id_here
   GITHUB_TOKEN=your_github_token_here  (optional)

4. INSTALL DEPENDENCIES
   $ pip install -r requirements.txt

5. RUN THE SERVER
   $ uvicorn main:app --reload --port 8000
   
   Expected output:
   ─── VickyBot Startup ───────────────────────
   ✅ CLOUDFLARE_API_TOKEN loaded: cfx_abc...
   ✅ CLOUDFLARE_ACCOUNT_ID loaded: abc123...
   ✅ GITHUB_TOKEN loaded: ghp_def...
   [agent] GitHub cache warmed: 5 repos loaded
   ────────────────────────────────────────────

6. TEST THE AGENT
   → Open http://localhost:8000
   → Chat with VickyBot
   → Expected response time: <1 second
